### Prompt Arguments

| Argument      | Type         | Question / Description                                                            |
|---------------|--------------|-----------------------------------------------------------------------------------|
| workspacePath | autocomplete | Where would you like to add this form? <br/>=> e.g. `features/@app-core`          |
| formSchema    | autocomplete | Which zod / input schema should we use? <br/>=> e.g. `HealthCheckInput` / `new`   |
| formHookName  | text         | What should the form hook be named? <br/>=> `string` (e.g. `useSomeSchemaState`)  |

<div className="h-8" />

### Resulting File Changes

```bash
/{workspacePath}/ # <- e.g. 'features/@app-core/' or 'packages/some-package/'
 └── hooks/
     └── {formHookName}.ts ❇️ # <- Will integrate with chosen / new schema
```
